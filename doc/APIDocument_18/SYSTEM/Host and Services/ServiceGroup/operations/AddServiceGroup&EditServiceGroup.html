Operation: Add Service Group / Edit Service Group
Description: To Create/Edit Service Group. 

Sample Configuration
<ServiceGroup> <Name>Name</Name> <Description>Text</Description> <ServiceList> <Service>ServiceName</Service> <Service>ServiceName</Service> <Service>ServiceName</Service> <Service>ServiceName</Service> <Service>ServiceName</Service> </ServiceList> </ServiceGroup>



Parameter Mandatory Default Description
NameYes  
Specify a name to identify the Service Group.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 60.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Specify Description of the Service Group.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
ServiceYes  
Select Services from the list to be added in the Service Group.
Service confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
