Operation: Update Central Management
Description: Manage and monitor the appliance through Central Management. 

Sample Configuration
<CentralManagement> <ManageUsingCentralManagement>Enable/Disable</ManageUsingCentralManagement> <ManagementType>SFM/SCFM</ManagementType> <ServerAddress>ip</ServerAddress> <ApplianceManagement> <ConfigurationManagement>Enable/Disable</ConfigurationManagement> <CommunicationDetails> <HeartbeatProtocol>Syslog/HTTPS</HeartbeatProtocol> <HeartbeatPort>514</HeartbeatPort> </CommunicationDetails> <ConfigurationSyncronization> <Mode>Push/Pull</Mode> <Port>443</Port> </ConfigurationSyncronization> </ApplianceManagement> <SignatureUpdates> <SignatureDistribution>Enable/Disable</SignatureDistribution> <Port>80</Port> </SignatureUpdates> </CentralManagement>



Parameter Mandatory Default Description
ManageUsingCentralManagementNo  
Enable to manage the appliance through Central Management.
ManageUsingCentralManagement confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'Enable' are allowed.
ManagementTypeNo  
Enable to manage the appliance through SFM or CFM.
ManagementType confines to:
  • Type is 'SCALAR'.
  • Only 'SCFM', 'SFM' are allowed.
ServerAddressNo  
Specify the IPv4 address/domain for Central Management.
ServerAddress confines to:
  • Type is 'SCALAR'.
  • Datatype is 'IPADDRESS','DOMAIN','IPADDRESS6'.
  • Maximum characters allowed are 255.
  • IP Class other than 'MULTICAST', 'RESERVED', 'LOCALHOST', 'UNSPECIFIED', 'BROADCAST', 'LINKLOCAL' is allowed.
HeartbeatProtocolNo  
Select Heartbeat protocol from the options available. It specifies how the information will be provided to Central Management.
HeartbeatProtocol confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
HeartbeatPortNo 80 
Specify the Heartbeat port. This is the port used for Keep Alive requests used by Central Management to check if Appliance is up.
HeartbeatPort confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Allowed port range: 1 to 65535
  • Maximum digits allowed are 5.
ModeNo  
Select the mode for sending configuration updates.
Mode confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
ProtocolNo  
Select connection protocol for sending updates.
Protocol confines to:
  • Type is 'SCALAR'.
  • Only 'HTTPS', 'HTTP' are allowed.
PortNo  
If Central Management is selected for receiving Signature Updates, Specify Central Management Port.
Port confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Allowed port range: 1 to 65535
  • Maximum digits allowed are 5.
SecureCommunicationNo Disable 
Enable to ensure secure communication between Appliance and Central Management.
SecureCommunication confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'false' are allowed.
PortNo  
Specify the port number of the selected connection protocol.
Port confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Allowed port range: 1 to 65535
  • Maximum digits allowed are 5.
