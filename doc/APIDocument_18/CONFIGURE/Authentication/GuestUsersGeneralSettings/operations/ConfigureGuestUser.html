Operation: Configure Guest User
Description: Configure parameters to allow secured internet access for guest users. 

Sample Configuration
<GuestUserSettings> <AllowGuestUserSettings>Enable/Disable</AllowGuestUserSettings> <SMSGateway>smsgatewayname</SMSGateway> <GuestUserSettingsName>CellNumber/AutoGenerate</GuestUserSettingsName> <grouplist_cat>Text</grouplist_cat> <smsgwprofileid_cat>Text</smsgwprofileid_cat> <!-- For AutoGenerate --> <UserNamePrefix>Username Prefix</UserNamePrefix> <UserValidity> <!-- Only one of below will be present --> <Days>Number</Days> </UserValidity> <AutoPurgeOnExpiry>Enable/Disable</AutoPurgeOnExpiry> <UserGroup>usergroupname</UserGroup> <CountryCode>country code</CountryCode> <CAPTCHVerification>Enable/Disable</CAPTCHVerification> <PasswordLength>Number</PasswordLength> <PasswordComplexity>AlphanumericPassword/AlphabeticPassword/NumericPassword/AlphaNumericWithSpecialCharacterPassword</PasswordComplexity> <Disclaimer>Text</Disclaimer> </GuestUserSettings>



Parameter Mandatory Default Description
AllowGuestUserSettingsNo  
Enable to allow secured internet access to guest users.
AllowGuestUserSettings confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
SMSGatewayYes  
Select the gateway to be used for sending SMS.
SMSGateway confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
GuestUserSettingsNameYes  
Select the method for generating Username from the available options.
GuestUserSettingsName confines to:
  • Type is 'SCALAR'.
  • Only 'CellNumber', 'AutoGenerate' are allowed.
UserNamePrefixYes  
Specify Prefix to be used for Auto-Generation of Username.
UserNamePrefix confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
DaysYes 7 
Specify guest user validity period in days.
Days confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Maximum digits allowed are 3.
AutoPurgeOnExpiryNo  
Enable automatic purging of user details on expiry of user validity.
AutoPurgeOnExpiry confines to:
  • Type is 'SCALAR'.
  • Only 'Enable' are allowed.
UserGroupYes  
Select Group to assign to the User.
UserGroup confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
CountryCodeNo  
Enable to configure default Country Code.
CountryCode confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
CAPTCHVerificationNo Enable 
Enable/Disable CAPTCHA code verification to protect against attacks generated by automated programs.
CAPTCHVerification confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'Disable' are allowed.
PasswordLengthNo 8 
Specify the length of auto-generated Password.
PasswordLength confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 3 to 50 is allowed.
  • Maximum digits allowed are 2.
PasswordComplexityNo AlphanumericPassword 
Select the type of complexity for the auto-generated Password.
PasswordComplexity confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 1 to 99 is allowed.
