Operation: Add OTP Token / Update OTP Token
Description: Add OTP TokenUpdate OTP Token 

Sample Configuration
<OTPTokens> <tokenid /> <timeStep /> <timeStepOffset /> <secret /> <lastLogin /> <timeOffset /> <extraCodes /> <active /> <autoCreated /> <user /> <comment /> </OTPTokens>



Parameter Mandatory Default Description
tokenidNo  
Specify 'tokenid'
tokenid confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 32.
  • Minimum characters allowed are 1.
timeStepNo  
OTP time step
timeStep confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 10 to 300 is allowed.
  • Maximum digits allowed are 3.
timeStepOffsetNo  
OTP time step offset
timeStepOffset confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
secretNo  
OTP Secret
secret confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 120.
  • Minimum characters allowed are 32.
lastLoginNo  
OTP last successful login
lastLogin confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
timeOffsetNo  
Time offset to adjust clock skew
timeOffset confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INT'.
extraCodesNo  
one-time random codes that may be used if the secret is temporarily unavailable
extraCodes confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 69.
activeNo  
indicates if is this token has been enabled or disabled
active confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
autoCreatedNo  
indicates if this token has been generated automatically at token creation
autoCreated confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
useridNo  
Specify 'userid'
confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
commentNo  
Tokens comment
comment confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
