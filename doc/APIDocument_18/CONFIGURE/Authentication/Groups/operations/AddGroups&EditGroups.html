Operation: Add Groups / Edit Groups
Description: To Create/Edit Groups. Group is a collection of policies that can be applied to the users by simply assigning the appropriate group to the users. 

Sample Configuration
<UserGroup> <GroupDetail> <Name>Name</Name> <After><Name>Name</Name></After> <GroupType>Normal/Clienless</GroupType> <!-- For Normal Group Type Configuration Parameters are as below --> <SurfingQuotaPolicy>SurfingQuota</SurfingQuotaPolicy> <AccessTimePolicy>AccessTime</AccessTimePolicy> <DataTransferPolicy>None</DataTransferPolicy> <QoSPolicy>None</QoSPolicy> <SSLVPNPolicy>No Policy Applied</SSLVPNPolicy> <ClientlessPolicy>No Policy Applied</ClientlessPolicy> <QuarantineDigest>Enable/Disable</QuarantineDigest> <MACBinding>Enable/Disable</MACBinding> <L2TP>Enable/Disable</L2TP> <PPTP>Enable/Disable</PPTP> <LoginRestriction>AnyNode/SelectedNodes/NodeRange</LoginRestriction> <!-- For SelectedNodes --> <NodeList> <IPAddress>IPAddress</IPAddress> : </NodeList> <!-- For Node Range --> <FromIP>ip</FromIP> <ToIP>ip</ToIP> <!-- For Normal Group Type Configuration Parameters ends --> <!-- For Clientless Group Type Configuration Parameters are as below --> <QoSPolicy>None</QoSPolicy> <!-- For Clientless Group Type Configuration Parameters ends --> </GroupDetail> <GroupMembers> <GroupName>Name</GroupName> <UserName>admin</UserName> : : </GroupMembers> </UserGroup>



Parameter Mandatory Default Description
NameYes  
Specify a name to identify the group.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 256.
GroupTypeNo  
Select Group type from the available options: Normal or Clientless.
GroupType confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
WebFilterPolicyYes  
Select Web filter policy from the list.
WebFilterPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
ApplicationFilterPolicyYes  
Select Application filter policy from the list.
ApplicationFilterPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
SurfingQuotaPolicyNo  
Select the Surfing Quota policy from the list.
SurfingQuotaPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
AccessTimePolicyNo  
Select the Access Time policy from the list.
AccessTimePolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
DataTransferPolicyNo  
Select the Data Transfer policy from the list.
DataTransferPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
QoSPolicyYes  
Select the QoS policy from the list.
QoSPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
ClientlessPolicyNo  
Select Clientless policy from the list.
ClientlessPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
QuarantineDigestNo Disable 
Enable to send Quarantine digest daily to the user which is an email containing a list of quarantined spam messages filtered by the appliance.
QuarantineDigest confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
MACBindingNo Disable 
Enable to bind user with a group of MAC Addresses.
MACBinding confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'Disable' are allowed.
L2TPNo Disable 
Enable if group users can get access through L2TP connection.
L2TP confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'Disable' are allowed.
PPTPNo Disable 
Enable if group users can get access through PPTP connection.
PPTP confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'Disable' are allowed.
LoginRestrictionYes Any Node 
Select appropriate option for user login restriction.
LoginRestriction confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
FromIPYes  
If Node Range option is selected for Login Restriction, Specify the starting IPv4 Address for the range between which the users will be allowed to login.
FromIP confines to:
  • Type is 'SCALAR'.
  • Datatype is 'IPADDRESS'.
ToIPYes  
If Node Range option is selected for Login Restriction, Specify the ending IPv4 Address for the range between which the users will be allowed to login.
ToIP confines to:
  • Type is 'SCALAR'.
  • Datatype is 'IPADDRESS'.
NameNo  
Specify 'moveto'
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
SSLVPNPolicyNo  
Select SSL VPN policy from the list.
SSLVPNPolicy confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
