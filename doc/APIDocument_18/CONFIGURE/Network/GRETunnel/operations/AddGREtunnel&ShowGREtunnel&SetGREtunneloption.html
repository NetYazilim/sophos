Operation: Add GRE tunnel / Show GRE tunnel / Set GRE tunnel option
Description: Add the GRE tunnelTo show the configuration of the GRE tunnelSet the GRE tunnel configuration for DDNS, tunnel state, and TTL 

Sample Configuration
<GreTunnel> <TunnelName>gre023</TunnelName> <LocalGateway>PortB</LocalGateway> <RemoteGateway>DomainName/IpAddress</RemoteGateway> <LocalNet>14.25.3.29</LocalNet> <RemoteNet>18.25.26.3</RemoteNet> <TTL>(0-255)</TTL> <Dyndns>On/Off</Dyndns> <State>Enabled/Disabled</State> </GreTunnel>



Parameter Mandatory Default Description
TUNNELNAMENo  
Enter the tunnel name
TUNNELNAME confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • To separate words, use a space.
  • Maximum characters allowed are 15.
TTLNo  
Enter the TTL
TTL confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 255 is allowed.
StateNo  
To set the state of the tunnel, enter 'Enable' or 'Disable'
State confines to:
  • Type is 'SCALAR'.
  • To separate words, use a space.
  • Only 'Enabled', 'Disabled' are allowed.
DYNDNSNo  
Enter the dynamic DNS
DYNDNS confines to:
  • Type is 'SCALAR'.
  • To separate words, use a space.
  • Only 'On', 'Off' are allowed.
LOCALGATEWAYNo  
Local gateway
LOCALGATEWAY confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • UTF-8 character(s) are allowed.
REMOTEGATEWAYNo  
Enter the remote WAN IP address or DDNS
REMOTEGATEWAY confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Allowed IPv4 address range: 0-255.0-255.0-255.0-255
  • To separate words, use a dot (.).
  • Maximum characters allowed are 64.
LOCALNETNo  
Enter the tunnel's local IP address
LOCALNET confines to:
  • Type is 'SCALAR'.
  • Datatype is 'IPADDRESS'.
REMOTENETNo  
Enter the remote IP address
REMOTENET confines to:
  • Type is 'SCALAR'.
  • Datatype is 'IPADDRESS'.
