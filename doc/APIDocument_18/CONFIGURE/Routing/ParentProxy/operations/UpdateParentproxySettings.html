Operation: Update Parentproxy Settings
Description: Update Parent Proxy settings when web traffic is blocked by upstream gateway.  

Sample Configuration
<ParentProxy> <ParentProxy>Enable/Disable</ParentProxy> <!-- For ParentProxy Enable Below configuration is necessary --> <ProxyServerAddress>1.1.1.1</ProxyServerAddress> <Port>3128</Port> <Username>test</Username> <Password>test</Password> <IPFamily>IPv4/IPv6</IPFamily> </ParentProxy>



Parameter Mandatory Default Description
ParentProxyNo  
Click to enable the Parent proxy if web traffic is blocked by upstream gateway.
ParentProxy confines to:
  • Type is 'SCALAR'.
  • Only 'Enable' are allowed.
ProxyServerAddressYes  
Specify Domain Name or IPv4 Address for the Parent Proxy.
ProxyServerAddress confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 45.
PortYes 3128 
To set Port number to be used for Parent Proxy.
Port confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Allowed port range: 1 to 65535
  • Maximum digits allowed are 5.
UsernameNo  
Specify Username for authentication.
Username confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 50.
  • UTF-8 character(s) are allowed.
PasswordNo  
Specify Password for authentication.
Password confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 50.
IPfamilyYes  
Used to determine whether the IP type is IPv4 or IPv6.
IPfamily confines to:
  • Type is 'SCALAR'.
  • Only 'IPv4', 'IPv6' are allowed.
