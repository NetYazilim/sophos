Operation: Add SD-WAN policy route / Update SD-WAN policy route
Description: Add an SD-WAN policy routeUpdate an SD-WAN policy route 

Sample Configuration



Parameter Mandatory Default Description
NameYes  
Specify a name for policy route.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 60.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Enter a description for the SD-WAN policy route.
Description confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 255.
IPFamilyYes IPv4 
Select the IP family.
IPFamily confines to:
  • Type is 'SCALAR'.
  • Only 'IPv4', 'IPv6' are allowed.
NetworkNo  
Select the source network(s) of the packet to be routed.
Network confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
NetworkNo  
Select the destination network(s) of the packet to be routed.
Network confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
ServiceNo  
Select the services(s) of the packet to be routed.
Service confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
UserNo  
Select the users and groups from the list.
User confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 256.
  • Multiple values are allowed.
BackupGatewayNo  
Select the backup gateway to which you want to forward the packet if the packet meets the configured matching criteria.
BackupGateway confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
InterfaceNo  
Select the incoming interface receiving the packet.
Interface confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
DSCPMarkingNo  
Select the type of DSCP Marking to match the packets marked with the given DSCP value.
DSCPMarking confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Range 1 to 64 is allowed.
HealthcheckNo  
Turn on health check to monitor the SD-WAN policy route.
Healthcheck confines to:
  • Type is 'SCALAR'.
  • Only 'OFF', 'ON' are allowed.
ApplicationObjectNo  
Select the application objects.
ApplicationObject confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
  • Multiple values are allowed.
GatewayYes  
Select the gateway to which you want to forward the packet if the packet meets the configured matching criteria.
Gateway confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
