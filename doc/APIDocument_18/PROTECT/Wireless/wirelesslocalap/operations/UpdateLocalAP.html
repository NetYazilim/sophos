Operation: Update Local AP
Description: The page allows one to update Local AP. 

Sample Configuration
<WirelessLocalAP> <Id>LocalWifi0/LocalWifi1</Id> <Country>In</Country> <WirelessNetworks> <Network>wlnet1</Network> <Network>wlnet2</Network> </WirelessNetworks> <BridgeToEthernet>Enable</BridgeToEthernet> <BridgeToPort>eth1</BridgeToPort> <Zone>LAN</Zone> <Band>2.4GHz/5GHz</Band> <Channel2.4GHz>Auto/1/2/3</Channel2.4GHz> <DynChan>Enable</DynChan> <TimeBasedScan>Enable</TimeBasedScan> <ScanTime> <Time>All Time on Weekends</Time> </ScanTime> <TXPower>100</TXPower> </WirelessLocalAP>



Parameter Mandatory Default Description
IdYes  
Displays ID of the Access Point.
Id confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
LabelNo  
Specify label
CountryYes  
Select Country where the Access Point is located.
Country confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
NetworkNo  
Select Wireless Networks for the Access Point.
Network confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
  • Duplicate values will not be ignored.
BridgeToEthernetYes  
Enable to bridge the wireless networks to Ethernet
BridgeToEthernet confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'Disable' are allowed.
Note:
Applicable only for the wireless networks that are of type, Bridge To LAN..
DynChanNo Disable 
Select this to enable Access Point to scan for channels dynamically.(5 GHz)
DynChan confines to:
  • Type is 'SCALAR'.
  • Only 't' are allowed.
TimeBasedScanNo Disable 
Select this to enable Time Based scan for Access Point.(2.4 GHz)
TimeBasedScan confines to:
  • Type is 'SCALAR'.
  • Only 't' are allowed.
Note:
Applicable only if Dyn Chan is enabled..
ChannelYes  
Select a channel that will be used by the access point.(2.4 GHz)
Channel confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
TXPowerYes 100% 
Select the transmission power for the Access Point.
TXPower confines to:
  • Type is 'SCALAR'.
  • Only '10%', '25%', '50%', '75%', '100%' are allowed.
BridgeToPortNo  
Select Port to create bridge with wireless network.
BridgeToPort confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
Note:
The interface must have an IP address and should not belong to the WAN port or any other bridge..
ZoneNo  
Select Zone for bridge connection
Zone confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
BandYes  
Select Local AP band , either 2.4Ghz or 5Ghz
Band confines to:
  • Type is 'SCALAR'.
  • Only '5GHz', '2.4GHz' are allowed.
TimeNo  
Select Scan Time for the Access Point(2.4 GHz).
Time confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
Note:
Applicable only if 'Time Based scan'(2.4 GHz) is enabled..
