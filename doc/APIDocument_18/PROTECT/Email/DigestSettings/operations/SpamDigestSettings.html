Operation: Spam Digest Settings
Description: Configure Quarantine Digest Settings for mailing Digests to the user as per the configured frequency. 

Sample Configuration
<AntiSpamQuarantineDigestSettings> <SpamDigestSettings> <SpamQuarantineDigest>Enable/Disable</SpamQuarantineDigest> <EmailFrequency>Hourly/Day/Week</EmailFrequency> <SendEmailAt> <Hour>Number</Hour> <Minute>Number</Minute> <Days>Sunday</Days> : : <Days>Saturday</Days> </SendEmailAt> <FromEmailAddress>email</FromEmailAddress> <DisplayName>SpamDigest</DisplayName> <AllowUserToOverride>Enable/Disable</AllowUserToOverride><!-- Allow User To Override Digest Settings --> <ReleaseLinkSettings>Interface/AdminHost</ReleaseLinkSettings> <ReleaseLinkInterface>PortA</ReleaseLinkInterface> <ReleaseLinkHost>abcd.local</ReleaseLinkHost> <QuarantineArea> <DiskSize>5GB/10GB/15GB</DiskSize> </QuarantineArea> </SpamDigestSettings> <SpamDigestUsers> <EnableUsers> <EnableUser>username</EnableUser> : : <EnableUser>username</EnableUser> </EnableUsers> <DisableUsers> <DisableUser>username</DisableUser> : : <DisableUser>username</DisableUser> </DisableUsers> </SpamDigestUsers> </AntiSpamQuarantineDigestSettings>



Parameter Mandatory Default Description
SpamQuarantineDigestNo  
Enable Quarantine Digest to configure Digest service.
SpamQuarantineDigest confines to:
  • Type is 'SCALAR'.
  • Only 'Enable', 'off' are allowed.
EmailFrequencyYes  
Specify the frequency of Quarantine Digest mails.
EmailFrequency confines to:
  • Type is 'SCALAR'.
  • Only 'Hourly', 'Day', 'Week' are allowed.
HourYes  
Select time interval in hours for Hourly Digest mails.
Hour confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 23 is allowed.
HourYes  
Select Hour of the Day for Daily Digest mails.
Hour confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 23 is allowed.
MinuteYes  
Select Minutes of the Hour for Daily Digest mails.
Minute confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 59 is allowed.
HourYes  
Select Hour of the Day for Weekly Digest mails.
Hour confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 23 is allowed.
MinuteYes  
Select Minutes of the Hour for Weekly Digest mails.
Minute confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 59 is allowed.
DaysYes  
Select Days for Weekly Digest mails.
Days confines to:
  • Type is 'ARRAY'.
  • Only 'Saturday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Sunday' are allowed.
  • Multiple values are allowed.
FromEmailAddressYes  
Specify Email Address from which Digest mails should be sent.
FromEmailAddress confines to:
  • Type is 'SCALAR'.
  • Datatype is 'EMAIL'.
  • Maximum characters allowed are 128.
DisplayNameYes  
Specify Mail Sender name.
DisplayName confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 100.
AddresspatternNo  
Specify address patterns for which the quarantine report will not be sent.
Addresspattern confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • EMAILWILDCARD
  • Multiple values are allowed.
ReleaseLinkSettingsNo  
To specify the quarantine release link settings, select the interface IP address or the hostname. You can change the settings in "Admin console and end-user interaction".
ReleaseLinkSettings confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
MailQuarantineSizeYes  
Specify the size of the quarantine area for emails.
MailQuarantineSize confines to:
  • Type is 'SCALAR'.
  • Only '$ANTISPAM{QUARANTINE_SIZE_FIVE}', '$ANTISPAM{QUARANTINE_SIZE_TEN}', '$ANTISPAM{QUARANTINE_SIZE_FIFTEEN}' are allowed.
