Operation: Schedule Backup / Schedule Backup Now
Description: Settings for scheduling backup.Settings for scheduling backup Now 

Sample Configuration
<BackupRestore> <ScheduleBackup> <BackupMode>FTP/Mail/Local</BackupMode> <BackupPrefix>Test</BackupPrefix> <FTPServer>1.1.1.1</FTPServer><!-- For FTP --> <Username>Test</Username><!-- For FTP --> <Password>Test</Password><!-- For FTP --> <FtpPath>/usr/backup</FtpPath><!-- For FTP --> <EmailAddress>test@test.com</EmailAddress><!--For Mail --> <BackupFrequency>Never/Daily/Weekly/Monthly</BackupFrequency> <Day>Sunday</Day> <Hour>01</Hour> <Minute>30</Minute> <Date>12</Date> <EncryptionPassword>Test</EncryptionPassword> </ScheduleBackup> </BackupRestore> <BackupRestore_Now> <ScheduleBackup> <BackupMode>FTP/Mail/Local</BackupMode> <BackupPrefix>Test</BackupPrefix> <FTPServer>1.1.1.1</FTPServer><!-- For FTP --> <Username>Test</Username><!-- For FTP --> <Password>Test</Password><!-- For FTP --> <FtpPath>/usr/backup</FtpPath><!-- For FTP --> <EmailAddress>test@test.com</EmailAddress><!--only if mail --> <BackupFrequency>Never/Daily/Weekly/Monthly</BackupFrequency> <Day>Sunday</Day> <Hour>01</Hour> <Minute>30</Minute> <Date>12</Date> <EncryptionPassword>Test</EncryptionPassword> </ScheduleBackup> </BackupRestore_Now>



Parameter Mandatory Default Description
BackupPrefixNo  
Prefix of Backup file
BackupPrefix confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 32.
BackupModeNo  
Select how and to whom the backup files are to be sent.
BackupMode confines to:
  • Type is 'SCALAR'.
  • Only 'FTP', 'Mail', 'Local' are allowed.
FTPServerYes  
Specify FTP Server IP address if backup is to be stored on FTP Server.
FTPServer confines to:
  • Type is 'SCALAR'.
  • Datatype is 'IPADDRESS','IPADDRESS6'.
  • Maximum characters allowed are 45.
  • IP Class other than 'MULTICAST', 'RESERVED', 'LOCALHOST', 'UNSPECIFIED', 'BROADCAST', 'LINKLOCAL' is allowed.
UsernameYes  
Specify Username if backup is to be stored on FTP Server.
Username confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 60.
  • UTF-8 character(s) are allowed.
PasswordNo  
Specify Password if backup is to be stored on FTP Server.
Password confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
EmailAddressYes  
Specify Email ID on which the backup is to be mailed.
EmailAddress confines to:
  • Type is 'CSV'.
  • Datatype is 'EMAIL'.
  • Comma separated values are allowed.
  • Maximum characters allowed are 300.
DateYes  
DATE
Date confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 1 to 31 is allowed.
MinuteYes  
Minutes
Minute confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 59 is allowed.
Password for backup encryptionNo  
Specify password to encrypt backup
Password for backup encryption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 1023.
  • Minimum characters allowed are 12.
DayYes  
weekday of Backup file
Day confines to:
  • Type is 'SCALAR'.
  • Only 'Sunday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Monday' are allowed.
FtpPathNo  
ftppath of Backup file
FtpPath confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 1024.
  • UTF-8 character(s) are allowed.
  • Character forward slash (/) is not allowed.
BackupFrequencyNo  
Select system data backup frequency from the options available.
BackupFrequency confines to:
  • Type is 'SCALAR'.
  • Only 'Never', 'Monthly', 'Weekly', 'Daily' are allowed.
HourYes  
Hour
Hour confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 0 to 23 is allowed.
