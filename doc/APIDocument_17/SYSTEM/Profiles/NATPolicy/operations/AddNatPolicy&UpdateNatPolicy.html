Operation: Add Nat Policy / Update Nat Policy
Description: To Create/Update NAT Policy. 

Sample Configuration
<NATPolicy> <Name>Name</Name> <MapSourceIP>HostName</MapSourceIP> </NATPolicy>



Parameter Mandatory Default Description
NameYes  
Specify a name to identify NAT Policy.
Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 30.
  • UTF-8 character(s) are allowed.
MapSourceIPYes  
Select IP Address to which the Source IP Address of the packet will be mapped.Host of type IP,IP Range are allowed.
MapSourceIP confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
