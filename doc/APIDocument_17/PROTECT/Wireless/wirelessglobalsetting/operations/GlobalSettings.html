Operation: Global Settings
Description:  

Sample Configuration
<WirelessProtectionGlobalSettings> <WirelessProtection>Enable</WirelessProtection> <AllowedZone> <Zone>LAN</Zone> <Zone>WiFi</Zone> </AllowedZone> <NotificationTimeout>5</NotificationTimeout> <RADIUSServer>None/Redius1</RADIUSServer> </WirelessProtectionGlobalSettings>



Parameter Mandatory Default Description
EnableWirelessProtectionNo  
Specify 'isenable'
EnableWirelessProtection confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
ZoneNo  
Specify 'allowedzone'
Zone confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
NotificationTimeoutNo  
Specify 'notification_timeout'
NotificationTimeout confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
RADIUSServerYes  
Specify 'radius_server'
RADIUSServer confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 50.
  • UTF-8 character(s) are allowed.
