Operation: Update Access Point
Description: The page allows one to update Access Points. 

Sample Configuration
<WirelessAccessPoint> <ID>A40016ADB63B7F1</ID> <Label>AP30[A40016ADB63B7F1]</Label> <Country>India</Country> <Group>No Group/wlgroup</Group> <WirelessNetworks> <Network>wlnet1</Network> <Network>wlnet2</Network> </WirelessNetworks> <MeshNetwoks> <MeshNetwok> <MeshID>mesh1</MeshID> <Role>RootAccessPoint</Role> </MeshNetwok> <MeshNetwok> <MeshID>mesh2</MeshID> <Role>RootAccessPoint</Role> </MeshNetwok> </MeshNetwoks> <Channel2.4GHz>Auto/1/2/3</Channel2.4GHz> <DynChan>Enable</DynChan> <TimeBasedScan>Enable</TimeBasedScan> <ScanTime> <Time>All Time on Weekends</Time> </ScanTime> <TXPower>100%</TXPower> <Channel5GHz>Auto/1/2/3</Channel5GHz> <DynChan5GHz>Enable</DynChan5GHz> <TimeBasedScan5GHz>Enable</TimeBasedScan5GHz> <ScanTime5GHz> <Time>All Time on Weekends</Time> </ScanTime5GHz> <TXPower5GHz>100%</TXPower5GHz> <STP>Enable/Disable</STP> <VLANTagging>Enable</VLANTagging> <APVLANID>25</APVLANID> </WirelessAccessPoint>



Parameter Mandatory Default Description
IDYes  
Displays ID of the Access Point.
ID confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
LabelYes  
Specify label
Label confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 30.
  • UTF-8 character(s) are allowed.
CountryYes  
Select Country where the Access Point is located.
Country confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
GroupNo  
Select Access point group for the Access Point.
Group confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
NetworkNo  
Select Wireless Networks for the Access Point.
Network confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
  • Duplicate values will not be ignored.
RoleNo  
Select role for the Access Point.
Role confines to:
  • Type is 'ARRAY'.
  • Only 'point', 'portal' are allowed.
  • Multiple values are allowed.
  • Duplicate values will not be ignored.
Channel2.4GHzNo  
Select a channel that will be used by the access point.(2.4 GHz)
Channel2.4GHz confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
DynChanNo Disable 
Select this to enable Access Point to scan for channels dynamically.(5 GHz)
DynChan confines to:
  • Type is 'SCALAR'.
  • Only 't' are allowed.
TimeBasedScanNo Disable 
Select this to enable Time Based scan for Access Point.(2.4 GHz)
TimeBasedScan confines to:
  • Type is 'SCALAR'.
  • Only 't' are allowed.
Note:
Applicable only if Dyn Chan is enabled..
ChannelWidthNo 20MHz 
Specify 'channel_width'
ChannelWidth confines to:
  • Type is 'SCALAR'.
  • Only '20', '40', '80' are allowed.
Note:
Applicable only in 2.4GHz band..
Channel5GHzNo  
Select a channel that will be used by the access point.(2.4 GHz)
Channel5GHz confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
DynChan5GHzNo Disable 
Select this to enable Access Point to scan for channels dynamically.(5 GHz)
DynChan5GHz confines to:
  • Type is 'SCALAR'.
  • Only 't' are allowed.
TimeBasedScan5GHzNo Disable 
Select this to enable Time Based scan for Access Point.(5 GHz)
TimeBasedScan5GHz confines to:
  • Type is 'SCALAR'.
  • Only 't' are allowed.
Note:
Applicable only if Dyn Chan is enabled..
TimeNo  
Select Scan Time for the Access Point(5 GHz).
Time confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
Note:
Applicable only if 'Time Based scan'(5 GHz) is enabled..
TXPower5GHzNo 100% 
Select the transmission power output for 5 GHz band.
TXPower5GHz confines to:
  • Type is 'SCALAR'.
  • Only '10%', '25%', '50%', '75%', '100%' are allowed.
STPNo Disable 
Select to use Spanning Tree protocol (STP).
STP confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
VLANTaggingNo Disable 
Select this to enable VLAN tagging and to connect Access Point with an existing VLAN Ethernet Interface.
VLANTagging confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
APVLANIDNo 2 
Specify the VLAN ID that will be used by the AP to connect to the appliance.
APVLANID confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
  • Range 2 to 4094 is allowed.
  • Maximum digits allowed are 4.
Note:
Available only when 'VLan Tagging' is selected..
MeshIDNo  
Select Mesh Networks for the Access Point.
MeshID confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
  • Duplicate values will not be ignored.
ChannelWidth11aNo 40MHz 
Specify 'channel_width11a'
ChannelWidth11a confines to:
  • Type is 'SCALAR'.
  • Only '20', '40', '80' are allowed.
Note:
Applicable only in 5GHz band..
TXPowerNo 100% 
Select the transmission power for the Access Point.
TXPower confines to:
  • Type is 'SCALAR'.
  • Only '10%', '25%', '50%', '75%', '100%' are allowed.
ACAbilityYes  
Specify 'band'
ACAbility confines to:
  • Type is 'SCALAR'.
  • Only '5GHz', '2.4GHz' are allowed.
TimeNo  
Select Scan Time for the Access Point(2.4 GHz).
Time confines to:
  • Type is 'ARRAY'.
  • Datatype is 'STRING'.
  • Multiple values are allowed.
Note:
Applicable only if 'Time Based scan'(2.4 GHz) is enabled..
