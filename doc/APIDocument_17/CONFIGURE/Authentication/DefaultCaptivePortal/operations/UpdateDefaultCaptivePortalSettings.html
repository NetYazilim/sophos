Operation: Update Default Captive Portal Settings
Description: To update default values used for captive portal. 

Sample Configuration
<DefaultCaptivePortal><!-- Values used for 'reset to default' button and uncustomizable fields --> <UserDefinedTemplate>HTML input</UserDefinedTemplate> <PageTitle>Sign in to access this network</PageTitle> <UsernameCaption>Username</UsernameCaption> <PasswordCaption>Password</PasswordCaption> <LoginButtonCaption>Login</LoginButtonCaption> <LogoutButtonCaption>Logout</LogoutButtonCaption> <UserPortalLinkCaption>Access the User Portal</UserPortalLinkCaption> <RegisterCaption>Register for internet access</RegisterCaption> <LoginPageHeader>HTML input</LoginPageHeader> <LoginPageFooter>HTML input</LoginPageFooter> <DoNotClosePage>Do not close this page</DoNotClosePage> <WillBeSignedOut>If you do, you will be signed out</WillBeSignedOut> <SigningYouIn>Signing you in...</SigningYouIn> <EnterUsername>Please enter your username.</EnterUsername> <EnterValidUsername>Please enter a valid username.</EnterValidUsername> <EnterPassword>Please enter your password.</EnterPassword> </DefaultCaptivePortal>
