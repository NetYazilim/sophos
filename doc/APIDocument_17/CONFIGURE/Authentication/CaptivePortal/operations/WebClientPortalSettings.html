Operation: Web Client Portal Settings
Description: Customize the Captive Portal page displayed for User Authentication.  

Sample Configuration
<CaptivePortal> <GeneralSettings> <Logo>Default/Custom</Logo> <FileName>Path of file</FileName><!-- for custom logo --> <LogoURL>http://www.organizationname.com</LogoURL> <PageTitle>Sign in to access this network</PageTitle> <LoginPageHeader>HTML input</LoginPageHeader><!-- HTML Input supported in this field--> <LoginPageFooter>HTML input</LoginPageFooter><!-- HTML Input supported in this field--> <UsernameCaption>Username</UsernameCaption> <PasswordCaption>Password</PasswordCaption> <LoginButtonCaption>Login</LoginButtonCaption> <LogoutButtonCaption>Logout</LogoutButtonCaption> <UserPortalLinkCaption>Access the User Portal</UserPortalLinkCaption> </GeneralSettings> <ColorScheme> <BackgroundColor>FFFFFF</BackgroundColor> <HeaderBackgroundColor>7D9ED1</HeaderBackgroundColor> <CaptionFontColor>000000</CaptionFontColor> <BackgroundFontColor>000000</BackgroundFontColor> <HeaderFontColor>07015C</HeaderFontColor> </ColorScheme> <FullCustomization> <EnableFullCustomization>True/False</EnableFullCustomization> <UserDefinedTemplate /> <SystemGeneratedHtml /> </FullCustomization> </CaptivePortal>



Parameter Mandatory Default Description
LogoNo  
Select 'Custom' if custom logo is to be uploaded or select 'Default'.
Logo confines to:
  • Type is 'SCALAR'.
  • Only 'Custom', 'Default' are allowed.
FileNameNo  
Browse the logo image to be displayed.
FileName confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • File formats 'jpg', 'gif', 'bmp', 'png', 'jpeg', 'jpe', 'jfif', 'dib', 'tif', 'tiff', 'JPG', 'GIF', 'BMP', 'PNG', 'JPEG', 'JPE', 'JFIF', 'DIB', 'TIF', 'TIFF' are allowed.
LogoURLNo http://www.cyberoam.com 
Specify URL associated with Logo.
LogoURL confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • The URL should start with ftp/http/https or the IP Address should be an IPv4 Address.
PageTitleNo Cyberoam Captive Portal 
Specify the Main Title of Captive Portal Page.
PageTitle confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 64.
LoginPageHeaderNo  
Specify content (as HTML) to be displayed as Header.
LoginPageHeader confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
LoginPageFooterNo  
Specify content (as HTML) to be displayed as Footer.
LoginPageFooter confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
UsernameCaptionNo Username 
Specify the caption or label for the 'Username' textbox.
UsernameCaption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 20.
PasswordCaptionNo Password 
Specify the caption or label for the 'Password' textbox.
PasswordCaption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 20.
LoginButtonCaptionNo Login 
Specify the caption or label for the Login Button.
LoginButtonCaption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 20.
LogoutButtonCaptionNo Logout 
Specify the caption or label for the Logout Button.
LogoutButtonCaption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 20.
UserPortalLinkCaptionNo Click here for User My Account 
Specify text to be displayed for My Account login page link.
UserPortalLinkCaption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 50.
BackgroundFontColorNo  
Set background Font color by specifying color code or picking color from the square box by clicking on it.
BackgroundFontColor confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 13.
BackgroundColorNo  
Set background color by specifying color code or picking color from the square box by clicking on it.
BackgroundColor confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 6.
HeaderBackgroundColorNo  
Set Header background color by specifying color code or picking color from the square box by clicking on it.
HeaderBackgroundColor confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 6.
HeaderFontColorNo  
Set Header Font color by specifying color code or picking color from the square box by clicking on it.
HeaderFontColor confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 6.
CaptionFontColorNo  
Set Caption Font color by specifying color code or picking color from the square box by clicking on it.
CaptionFontColor confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 6.
Register CaptionNo  
Use to get Username to access the internet.
Register Caption confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 70.
Html TextNo  
full customization html text.
Html Text confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
EnableFullCustomizationNo  
enable full customization.
EnableFullCustomization confines to:
  • Type is 'SCALAR'.
  • Only 'False', 'True' are allowed.
UserDefinedTemplateNo  
full customization html text.
UserDefinedTemplate confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
