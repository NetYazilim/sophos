Operation: Add SSLVPN Client Connection / Edit SSLVPN Client Connection
Description: To Add/Edit SSLVPN Client Connection.To Add/Edit SSLVPN Client Connection. 

Sample Configuration
<SiteToSiteClient> <Name>test</Name> <ServerConfigurationFile>{file upload in .apc or .epc format}</ServerConfigurationFile> <FilePassword>alphanumeric</FilePassword> <HttpProxyServer>Enable/Disable</HttpProxyServer> <!-- If HttpProxyServer is Enable --> <ProxyServer>ProxyServerName</ProxyServer> <ProxyPort>PortNumber</ProxyPort> <ProxyAuthentication>Enable/Disable</ProxyAuthentication> <!-- If ProxyAuthentication is Enable --> <Username>username</Username> <Password>password</Password> <PeerHost>Enable/Disable</PeerHost> <!-- If PeerHost is Enable --> <HostName>HostName</HostName> <Description>text</Description> <Status>On/Off</Status> </SiteToSiteClient>



Parameter Mandatory Default Description
Connection NameYes  
Enter a descriptive name for the connection.
Connection Name confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Character not allowed: Comma (,)
  • Maximum characters allowed are 50.
  • UTF-8 character(s) are allowed.
DescriptionNo  
Enter a description or other information.
Configuration FileYes  
Click the 'Browse...' icon to browse for the client configuration file. The file has to be in .apc or .epc format.
Configuration File confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
PasswordNo  
If the file has been encrypted, enter the password.
Password confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
Use HTTP Proxy ServerNo  
Enable/Disable to use HTTP as proxy server.
Use HTTP Proxy Server confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
Proxy ServerYes  
Select or add a proxy server.
Proxy Server confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
Proxy PortYes  
Enter a proxy port.
Proxy Port confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
Proxy Requires AuthenticationNo OFF 
Enable if the client needs to authenticate against the proxy.
Proxy Requires Authentication confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
UsernameNo  
Enter a username to authenticate against the proxy.
Username confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
PasswordNo  
Enter a password to authenticate against the proxy.
Password confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 60.
Override Peer HostnameNo Disable 
Enable/Disable to override peer hostname.
Override Peer Hostname confines to:
  • Type is 'SCALAR'.
  • Only 'Disable', 'Enable' are allowed.
HostnameYes Disable 
Enter a hostname if the server system's regular hostname cannot be resolved from the client host.
Hostname confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
StatusNo  
Status.
Status confines to:
  • Type is 'SCALAR'.
  • Only 'Off', 'On' are allowed.
