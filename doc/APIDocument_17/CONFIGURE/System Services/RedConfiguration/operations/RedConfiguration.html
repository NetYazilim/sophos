Operation: Red Configuration
Description: To register Sophos Firewall OS at the RED Provisioning Service (RPS) of Sophos to act as a RED hub. 

Sample Configuration
<RED> <REDConfiguration> <Status>Enable/Disable</Status> <OrganizationName /> <City /> <Country /> <Email /> </REDConfiguration> </RED>



Parameter Mandatory Default Description
StatusYes  
Enter the status.
Status confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
OrganizationNameNo  
Enter the name of the organization.
OrganizationName confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
CityNo  
Enter the city where the organization is located.
City confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
CountryNo  
Select the country where the organization is located.
Country confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
EmailNo  
Enter an email address.
Email confines to:
  • Type is 'SCALAR'.
  • Datatype is 'EMAIL'.
deauthorizationNo  
Turn off/on Auto-Deauthorization for RED device.
deauthorization confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
deauth_timeoutNo  
Enter time after a RED device has to be Auto-Deauthorized.
deauth_timeout confines to:
  • Type is 'SCALAR'.
  • Datatype is 'INTEGER'.
ForceTLS12No  
Enable/Disable the forcing of TLS1.2
ForceTLS12 confines to:
  • Type is 'SCALAR'.
  • Only '0', '1' are allowed.
RunBetaFirmwareNo  
Run RED beta firmware
RunBetaFirmware confines to:
  • Type is 'SCALAR'.
  • Datatype is 'STRING'.
  • Maximum characters allowed are 32.
